# Color Print
This is a command line project for those who want to add some color to their
bash scripts.
## Getting Started
### prerequisites
cmake
### installation
```
mkdir build
cd build
cmake ../
make
make install # if you want global installation (might need sudo privileges)
```
### basic usage
```
echo "INFO: check this out" | clrprint
```
